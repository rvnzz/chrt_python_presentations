# Что такое условный оператор?

#### **Условный оператор** — конструкция, которая позволяет выполнять разный код в зависимости от условия

::right::

## Базовый синтаксис:

```python
if условие:
    # код выполняется, если условие истинно
    действие
```

---

# Оператор if

##### Используется для проверки условия, код выполнится если условие будет положительным.

```python
age = 18
if age >= 18:
    print("Вы совершеннолетний")
```

## Важно:
- Условие должно возвращать `True` или `False`
- Код внутри блока **обязательно** имеет отступ (Tab)

---

# Оператор if-else

##### Выбор между двумя вариантами, если условие в `if` дало отрицательный результат, то выполнится код `else`


```python
age = 17
if age >= 18:
    print("Вы совершенолетний")
else:
    print("Вы ещё не совершенолетний")
```

---

# Оператор if-elif-else

#### Проверка нескольких условий, условия проверяются последовательно, сначала `if`, потом `elif`, если все условия ложные то выполняется `else`

```python
score = 85

if score >= 90:
    grade = "Отлично"
elif score >= 75:
    grade = "Хорошо"
elif score >= 50:
    grade = "Удовлетворительно"
else:
    grade = "Неудовлетворительно"

print(f"Ваша оценка: {grade}")
```

# Множественные условия с elif

```python
day = "понедельник"

if day == "суббота" or day == "воскресенье":
    print("Выходной день")
elif day == "понедельник":
    print("Начало рабочей недели")
elif day == "пятница":
    print("Последний рабочий день")
else:
    print("Обычный рабочий день")
```

# Операторы сравнения

- `==` - равно
- `!=` - не равно
- `>` - больше
- `<` - меньше
- `>=` - больше или равно
- `<=` - меньше или равно

# Логические операторы в условиях

```python
# Оператор and (И)
age = 25
has_ticket = True
if age >= 18 and has_ticket:
    print("Вы можете войти на концерт")

# Оператор or (ИЛИ)
is_weekend = True
is_holiday = False
if is_weekend or is_holiday:
    print("Можно отдыхать!")

# Оператор not (НЕ)
is_raining = False
if not is_raining:
    print("Можно идти гулять")
```

# Комбинирование логических операторов

```python
age = 25
income = 50000
credit_score = 720

# Проверка условий для кредита
if (age >= 21 and age <= 65) and (income >= 30000) and (credit_score >= 650):
    print("Кредит одобрен")
else:
    print("Кредит отклонён")

# Использование скобок для приоритета
if (age < 18 or age > 65) and income < 20000:
    print("Специальные условия")
```

# Проверка значений на истинность

```python
# Пустые значения = False
empty_string = ""
if empty_string:
    print("Строка не пустая")
else:
    print("Строка пустая")  # Выведет это

# Числа: 0 = False, всё остальное = True
value = 0
if value:
    print("Значение ненулевое")
else:
    print("Значение равно нулю")  # Выведет это
```
