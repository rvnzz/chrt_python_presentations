# Присваивание значений

```python
x = 10                   # простое присваивание

y = x + 5                # присваивание результата выражения

a, b, c = 1, 2, 3        # множественное присваивание
print(a, b, c)           # Выведет: 1 2 3

x, y = y, x              # обмен значениями
print(x, y)              # Выведет: 15 10
```

# Правила именования переменных

## Основные правила:

1. **Начинаются с буквы или подчеркивания** (`_`)
2. **Могут содержать**: буквы, цифры, подчеркивания
3. **НЕ могут начинаться с цифры**
4. **Чувствительны к регистру**: `name` ≠ `Name` ≠ `NAME`
5. **Нельзя использовать зарезервированные слова**

## Зарезервированные слова (keywords):

```python
False, None, True, and, as, assert, async, await,
break, class, continue, def, del, elif, else, except,
finally, for, from, global, if, import, in, is, lambda,
nonlocal, not, or, pass, raise, return, try, while, with, yield
```

# Базовые типы данных в Python

| Тип | Название | Описание | Пример |
|-----|----------|----------|--------|
| `int` | Целое число | Целые числа без ограничений | `42`, `-17`, `0` |
| `float` | Вещественное число | Числа с плавающей точкой | `3.14`, `-0.5`, `2.0` |
| `str` | Строка | Последовательность символов | `"Hello"`, `'Python'` |
| `bool` | Логический тип | Истина или Ложь | `True`, `False` |

# Методы строк

```python
# Изменение регистра
text = "Hello World"
print(text.upper())      # "HELLO WORLD"

# Поиск и замена
print(text.find("World"))  # 6 (индекс начала)
print(text.replace("World", "Python"))  # "Hello Python"

# Разделение и объединение
words = text.split()     # ['Hello', 'World']
joined = "-".join(words) # "Hello-World"

# Проверки
print(text.startswith("Hello"))  # True
print(text.isdigit())            # False
```

# Логический тип (bool)

```python
# Создание логических значений
is_active = True
is_valid = False

# Логические операции
result1 = True and False   # False
result2 = True or False    # True
result3 = not True         # False

# Операции сравнения возвращают bool
x, y = 10, 20
print(x == y)    # False
print(x < y)     # True

# Преобразование в bool
print(bool(0))       # False
print(bool(1))       # True
print(bool(""))      # False (пустая строка)
```

# Преобразование типов (Type Casting)

## Явное преобразование

```python
# Преобразование в int
x = int("123")        # 123
y = int(3.14)         # 3 (отбрасывается дробная часть)

# Преобразование в float
a = float("3.14")     # 3.14
b = float(5)          # 5.0

# Преобразование в str
s1 = str(100)         # "100"
s2 = str(3.14)        # "3.14"

# Преобразование в bool
print(bool(0))        # False
print(bool("text"))   # True
```

# Арифметические операции

| Операция | Синтаксис | Пример | Результат | Описание |
|----------|-----------|---------------------|-----------|----------|
| Сложение | `a + b` | `10 + 3` | `13` | Сумма двух чисел |
| Вычитание | `a - b` | `10 - 3` | `7` | Разность двух чисел |
| Умножение | `a * b` | `10 * 3` | `30` | Произведение двух чисел |
| Деление | `a / b` | `10 / 3` | `3.333...` | Деление (всегда float) |
| Целочисленное деление | `a // b` | `10 // 3` | `3` | Деление без остатка |
| Остаток от деления | `a % b` | `10 % 3` | `1` | Остаток при делении |
| Возведение в степень | `a ** b` | `10 ** 3` | `1000` | a в степени b |

## Приоритет операций

| Приоритет | Операции | Описание |
|-----------|----------|----------|
| 1 (высший) | `**` | Возведение в степень |
| 2 | `*`, `/`, `//`, `%` | Умножение и деление |
| 3 (низший) | `+`, `-` | Сложение и вычитание |

**Совет:** используйте скобки `()` для изменения порядка выполнения операций!

# Операции сравнения

| Оператор | Синтаксис | Пример | Результат | Описание |
|----------|-----------|---------------------|-----------|----------|
| Равно | x == y | `10 == 20` | `False` | Проверка на равенство |
| Не равно | x != y | `10 != 20` | `True` | Проверка на неравенство |
| Меньше | x < y | `10 < 20` | `True` | Строго меньше |
| Больше | x > y | `10 > 20` | `False` | Строго больше |
| Меньше или равно | x <= y | `10 <= 20` | `True` | Меньше либо равно |
| Больше или равно | x >= y | `10 >= 20` | `False` | Больше либо равно |

# Логические операции

## Основные операторы

| Оператор | Синтаксис | Пример | Результат | Описание |
|----------|-----------|---------|-----------|----------|
| И (AND) | `a and b` | `True and False` | `False` | Истина, если оба True |
| ИЛИ (OR) | `a or b` | `True or False` | `True` | Истина, если хотя бы один True |
| НЕ (NOT) | `not a` | `not True` | `False` | Инверсия значения |
